FashionGhar App Backend (Node.js E-commerce API)

This project is a Node.js backend for an e-commerce platform that powers the FashionGhar shopping application.

📖 About

A Node.js backend API for the FashionGhar e-commerce app that manages products, users, orders, and authentication.

🚀 Features
User authentication (JWT)
Product management (CRUD)
Order processing system
Cart functionality
Category management
Secure REST APIs
MongoDB integration
🏗️ Project Structure
FashionGhar-App-Backend
│
├── controllers
├── models
├── routes
├── middleware
├── config
└── server.js
⚙️ Tech Stack
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
